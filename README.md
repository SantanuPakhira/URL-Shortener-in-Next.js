# 🔗 URL Shortener

A lightweight and modern URL Shortener built using [Next.js](https://nextjs.org/), [JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript), and [Tailwind CSS](https://tailwindcss.com/).  
Easily shorten long URLs and share them with a clean, responsive frontend.

## 🚀 Features

- Shorten long URLs instantly
- Responsive UI with Tailwind CSS
- Copy to clipboard functionality
- Real-time input validation
- Fast page navigation with Next.js routing

## 🖼️ Tech Stack

- **Frontend**: Next.js, JavaScript, Tailwind CSS
- **backend**:next.js, PostMan, MongoDB

```bash
# Clone the repository

# Navigate into the project
cd url-shortener

# Install dependencies
